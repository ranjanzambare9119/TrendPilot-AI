# TrendPilot AI

TrendPilot AI is an AI-powered Social Media Assistant built using IBM watsonx Orchestrate and IBM Granite models.

## Features
- AI-powered social media assistance
- Content and caption generation
- Marketing recommendations
- Responsive web interface

## Technologies
- IBM watsonx Orchestrate
- IBM Granite
- IBM Cloud
- HTML, CSS, JavaScript
